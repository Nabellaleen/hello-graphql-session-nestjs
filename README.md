## Description

A demo implementation of authentication using NestJS + GraphQL + Session + Passport

** ! Actually not working due to session issues ! **

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
